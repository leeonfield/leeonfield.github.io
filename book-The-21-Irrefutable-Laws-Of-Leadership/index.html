读书-领导力 21 法则

2020年3月29日

20200323-20200329

信任和沟通，目标与合作。

  • 推荐：四星
  • 结论：这篇文章适合团队中的每个角色去看，思考团队的意义、价值，如何合作，达成目标。
  1. 盖子：目标的高度决定了未来的潜力
  2. 影响力：影响力的本质在于让别人参与其中
    1. 性格-他们是谁
    2. 关系-他们认识谁
    3. 知识-他们知道什么
    4. 直觉-他们感受到什么
    5. 经验-他们去过哪里
    6. 过去的成就-他们做过什么
    7. 能力-他们能做什么
  3. 过程：去积累，去实践计划
  4. 导航：制定长远计划，涉及维度事无巨细，考虑周全，最后要做的只是贯彻执行
    1. 行动计划
    2. 列出目标
    3. 设定优先次序
    4. 告知关键人物
    5. 执行
    6. 预测可能出现的问题
    7. 明确目标、方向
    8. 每日对计划进行回顾
  5. 增值：领导力的底线不在于自己能走多远，而在于能让别人走多远
  6. 根基：信任是领导力的根基，
    1. 让你关心的人看到你对 ta 的关心
    2. 每个决定之前必须走的基础流程：沟通、获得反馈
  7. 尊重：人们通常愿意追随比自己强的领导者
  8. 直觉：尝试用直觉去评估每件事
    1. 直觉的背后是对人的认识，对资源网络的梳理，达到胸有成竹的状态
    2. 上面一点可以通过学习达成
  9. 吸引力：你只能吸引和你相似的人
  10. 亲和力：融入到团队中，大家只是角色不同，身份都是团队的一员
    1. 展现真我
    2. 坦率真诚
    3. 了解你的团队成员
    4. 身体力行
    5. 身临其境，面对面沟通，用对方的语言来沟通
    6. 关注他们，而不是自己
    7. 信任
    8. 指明方向，带来希望
  11. 核心圈：团队与 leader 相辅相成，成员能力互补，相互成就
    1. 团队成员要求
      1. 新成员是否有影响力
      2. 能否给团队带来互补的才能
      3. 能否担任要职，独当一面
      4. 能否增加团队的价值
      5. 能否给团队成员带来积极影响
    2. 组团方式：鉴别->培养->吸收
    3. 团队需要迭代成长，leader 需要跟着团队的更迭一起成长
  12. 授权：让合适的人去做擅长的事，并克制自己过程中不横加干涉
    1. 授权的障碍
      1. 内心缺失工作上的安全感：自身的价值需要在更高的层次体现
      2. 抗拒改变
      3. 缺乏自我肯定
    2. 被授权者可以获得机会实现更多的目标，授权者也通过授权获得了更多更高的权力
  13. 镜像：言行一致，作为 leader，每个行为、决定都会被团队成员所观察，成员成员们的镜像对象，如果不能时刻遵守，会对团队成员造成困扰。
  14. 接纳：让团队成员接纳 leader，让团队成员接纳你成为领导者
    1. 领导 + 愿景 = 结果
      1. 不接纳、不采信，换个领导
      2. 不接纳、采信，换个领导
      3. 接纳、不采信，换个想法
      4. 接纳、采信，全新追随
    2. 获得接纳的方式
      1. 通过接触发展良好关系
      2. 表现城市，建立信任
      3. 提高自我标准，树立榜样
      4. 为团队成员完成工作提供帮助
      5. 帮助团队成员实现个人目标
      6. 培养他成为领导者
  15. 制胜：团队目标就是拿结果，这是领导者的使命
    1. 拿结果的前置条件
      1. 统一的目标，团队成员的目标需要一致，需要让他们知道团队的目标是什么
      2. 多样化的技能，团队成员各自擅长领域不同，在实现目标的过程中需要各展所长
      3. leader 需要把上面两件事做好，技能不同的人聚在一起，给他们找一个一直的目标，大家一起拿结果
  16. 动势： 书中关于动势这部分的描述我读起来有些虚幻，聊下我对这个关键词的看法，团队的发展需要不断有更高的目标来激励成员们长久保持激情的状态，风浪中，大船停滞不前就是在后退，作为 leader 需要给团队找到更高的目标，保持团队的战斗力，如果沉迷眼前的成就，士气不久就会低落。
  17. 优先次序：忙碌不一定等于有效，
    1. 优先次序的标准
      1. 分内的事情（requirement）
      2. 回报效益高的事情（return）
      3. 回报大的事情（reward）
    2. 实践
      1. 日常生活严格执行优先次序的标准
      2. 制定团队的优先次序标准
  18. 舍得：实现目标的过程中，需要学会放弃，这部分可以上一条优先次序一起看，拿结果的过程中，我们需要付出一些代价去换取优先级更高的目标。
  19. 时机：抓住机遇
    1. 时机和决策的关系
      1. 错误的时机 + 错误的决策 = 灾难
      2. 错误的时机 + 正确的行动 = 抵制
      3. 正确的时机 + 错误的行动 = 错误
      4. 正确的时机 + 正确的行动 = 成功
    2. 实践：我在看这部分内容之前对于时机和决策的看法是，需要经验，我认为需要足够的积累才能在合适的时机做出合适的决策，但这种方案成本很高，时间成本更高。但这部分内容给了我新的思考方向，即训练的不是经验，是思考模型，不如给自己的决策一个范式，贯彻执行并且不断去完善这个范式，思考方向可以量化，决策的结果考虑的维度也会更多，更周全
      1. 失败的原因思考：目标是什么、谁去做、哪些因素需要被考虑、谁有经验、团队士气如何、市场和行业如何、哪些资源可以借调使用、时间的早迟会对结果产生影响吗、为什么会失败
      2. 规划时思考：局势的认识、动机是否正确、行动是否有信心、能否果断决定、是否汲取已有的经验、直觉维度是否考虑在内，如士气和动势、做好准备了吗
  20. 爆炸新倍增：将能力复制，形成群体效应，做大
    1. 领导力阶段
      1. 第一阶段，提高自己
      2. 第二阶段，发展团队
      3. 第三阶段，培养领导者
  21. 传承： 领导者的价值由其继承者决定
    1. 清楚自己留下的遗产
    2. 实现自己希望留下的遗产
    3. 挑选事业继承人
    4. 确保顺利传递交接棒

团队中的粘合剂是信任，信任需要通过沟通去达成，和他们沟通团队的目标、他们的目标，做决定之前了解他们的想法，让他们知道“我”的想法，坦诚相待。

领导者是一个团队的灵魂所在，既要仰望太空，看到长远的未来，也要脚踏实地，组团队，协调人力、资源，协调团队成员的角色，缺一不可。
