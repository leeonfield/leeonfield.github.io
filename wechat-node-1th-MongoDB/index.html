结构

关于MongoDB的结构，官方文档给出了下面的参考

SQL MongoDB
database database
table collection
row document or BSON document
column field
index index
primary key primary key(默认自动设置为_id)

注：_id是在插入一条新的document时，系统自动生成的一个由时间、主机名、PID(进程ID)、INC(自增计数器)组合转换而成的24位字符串。
