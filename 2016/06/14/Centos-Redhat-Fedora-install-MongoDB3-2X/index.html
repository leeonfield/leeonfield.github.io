CentOS 7(64位)上使用yum安装MongoDB。

添加MongoDB的yum库

安装MongoDB

安装命令
# yum install mongodb-org

启动MongoDB

使用mongodb-org-server提供的初始化脚本启动服务
# /etc/init.d/mongod restart
设置开机启动MongoDB
# chkconfig mongod on

开始使用

使用mongo命令进入命令行模式，可以放肆的玩耍了
