TEST 语言的词法规:

  • 标识符：字母打头，后接任意字母或数字

  • 保留字：标识符的子集，包括：if,else,for,while,do, int,write,read，

  • 无符号整数：由数字组成，但最高位不能为0，允许一位的0，

  • 分界符：(、)、;、{、}

  • 运算符：+、-、*、/、=、<、>、>=、<=、!=、==

  • 注释符：/ /

正则表达式：

  • 标识符：        ( a|b|……|z|A|B……|Z )( 0|1|……|9| a|b|……|z|A|B……|Z )*

  • 保留字：        标识符的子集

  • 无符号整数： ( (1……|9 )( 0|1|……|9)* )|0

  • 分界符：        ( | ) | ; | { | }

  • 运算符：        + | - | * | / | = | < | > | >= | <= | != | ==

  • 注释符：        /* (other)* */
