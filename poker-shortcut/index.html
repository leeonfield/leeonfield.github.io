    • 一个半月前，夏部长的青轴出现了 P 键间歇失灵的问题，忍了一段时间就下单了 Poker 茶轴，用着很舒服，不过由于组合键太多，导致经常出现按键锁定的情况，每到这个时候，我就像个猴子在电脑前，按哪错哪。

    这里总结一些常见的问题，供大家快速解决问题

    TL;DR

    Q: 左 WIN 键亮灯怎么关 ？
    A: FN + Z

    Q: 空格左灯怎么关 ？
    A: FN + 右 Shift

    Q: 空格右灯怎么关 ？
    A: FN + 右 Ctrl

    Q: 怎么打出 ~ ？
    A: Shift + FN + ESC (注意顺序)

    Q: 只能打出大写字母／大写锁定了
    A: 再按一次 Shift，如果左 WIN 键亮灯，可以参照第一条关掉

    Q: 关掉键盘的 LED 灯 ？（仅限于带背光版本）
    A: FN + v

    Q: w、a、s、d 打不出来，变成方向键了
    A: FN + 空格键

    Q: PN 键被锁定了，锁定编程层才能正常使用（FN + 右 Shift）才能正常用键码
    A：键盘断电试试（亲测可行，可能是 bug）

    完整版指南

    常用功能

    FN 常用功能键 功能键 组合键
    FN ESC `~
    FN [1、2、3…… -> +] [F1 -> F12]
    FN BACKSPACE DEL
    FN I INS
    FN P PRINT SCREEN
    FN [ { SCROLL LOCK
    FN ] } PAUSE
    FN ENTER 副 ENTER
    FN ; : HOME
    FN ‘ “” PAGE UP
    FN X APP
    FN . > END
    FN / ? > PAGE DOWN
    FN W ↑
    FN A ←
    FN S ↓
    FN D →

    多媒体功能

    FN 多媒体功能键 功能键 组合键
    FN Y 计算器
    FN N 音量减
    FN M 音量增
    FN , < 静音 🔇

    LED 功能（适用于背光版本）

    LED 功能键 功能 备注
    FN V 全键 LED 亮（除左 WIN） 按压第一次
    FN V 关闭 LED 按压第二次
    FN C 降低 LED 亮度 共 7 级亮度
    FN B 增加 LED 亮度 共 7 级亮度
    FN E 保存背光亮度 无

    编程相关

    编程相关 功能键 组合键 备注
    FN R 复位键（慎重） 必须长按 12 秒
    FN 右 Shift 切换至编程层 按压第一次（空格左灯亮）
    FN 右 Shift 切换回编程层 按压第二次（空格左灯灭）
    FN 右 Ctrl 进入编程模式 按压第一次（PN 灯亮）
    FN 右 Ctrl 退出编程模式 按压第二次（PN 灯灭）
    FN F 延时 15 ms 仅在编程模式下有效
    FN G 延时 0.1 s 仅在编程模式下有效
    FN H 延时 0.5 s 仅在编程模式下有效

    编程举例

    目标：把 a 编为 Ctrl + FN + a （mac 下切换屏幕）

    1. FN + 右 Ctrl 打开编程模式，空格右键亮起
    2. 点下 a 键， 接着按下 Ctrl + FN + a 组合键，放开，再点击 PN 键保存编辑
    3. FN + 右 Ctrl 退出编程模式，空格右灯熄灭
    4. PN + a => 屏幕切换，成功
    5. 现在我们锁定编程模式 FN + 右 Shift，按 a => 屏幕切换 （get it ?)

    特殊功能

    FN 特殊功能 功能键 组合键 备注
    FN Z 把 Shift 键变成按下一直发码，再按一下停止发码 按压第一次（左 WIN 灯亮）
    FN Z Shift 键恢复正常 按压第二次（左 WIN 灯灭）
    FN SPACE W，A，S，D 转换为 ↑←↓→ 按压第一次
    FN SPACE ↑←↓→ 转换为 W，A，S，D 按压第二次

    指拨开关说明

    指拨开关说明 ON OFF 备注
    开关 1 CAP = 左 WIN / 左 FN
    CAP 灯 = 左 WIN 灯    		 CAP = CAP
    CAP 灯 = CAP 灯    		 预设 = OFF 
    开关 1 左 WIN / 左 FN = CAP
    左 WIN 灯 = CAP 灯    		 左 WIN / 左 FN = 左 WIN / 左 FN
    左 WIN 灯 = 左 WIN 灯    		 预设 = OFF 
    开关 2 右 Ctrl = `~ 右 Ctrl = 右 Ctrl 预设 = OFF 
    开关 3 左 WIN = 左 FN 左 WIN = 左 WIN 预设 = OFF
    开关 4 写保护键盘，按 FN + R
    或当尝试编程按键都会导致 PN 灯快速闪烁一秒
    然后熄灭并返回非编程模式的正常状态    		 预设 = OFF
