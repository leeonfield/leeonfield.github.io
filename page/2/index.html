Centos7安装Node以及更新

Posted on 2016-06-12   |   In 工具使用   |     |   visitors

安装node & npm

Yum安装
使用yum安装，需要先启用EPEL repository
# yum install epel-release

安装node
# yum install nodejs
查看是否安装成功
# node -version

安装npm
# yum install npm
查看是否安装成功
# npm -version

安装express

查看是否安装成功
express -version

更新

node更新

安装 n 模块
npm install -g n
通过 n 模块安升级到NodeJS最新稳定版
n stable

同时，可以变更到指定版本
n 0.10.26 //切换到版本 0.10.26
