llog-6-做确定的事情

2019年12月20日

22m

都知道该做什么，就是不去做。

今天下午偶然翻到一篇大约四年前看过的文章，文章的内容记得不大清晰了，只是时间看起来较为感慨，工作之后，生活开始进入快节奏，值得留下的记忆太少，多数都是日复一日的工作，甚至已经忘记了具体的时间，唯一记得的是下周的产品迭代的版本，只要进入了快节奏的生活，面对未来都自顾不暇，自然很少能想起回顾过往。一七年开始正式工作，还差两个月满算三年。不过具体到细节，突然什么都想不起来，只记得做了很多事情。不过，相比于刚毕业，两年多的锻炼让我获得了真正的成长，远超过在学校度过的十几年，但不能比较哪一种成长更有价值，每一阶段的价值都无可替代。只是这两年的经历，让我更接受自己，学会与自己相处。

我还是会偶尔去假设过往如何如何，但真给我重来的选项，我还是会犹豫，因为那些我认为的不堪也是我成长的一部分，我学会接受它们并且和平相处。无论是接受自己，或是开始自信，都是在潜移默化中转变，17 年秋天的我还在 llog 迷茫，不知道下一步怎么走，只知道做好眼前的工作、学习，周末到公司学习。未来是不确定的，至少眼前每天的成长是确定的。

对于正确事情的认知，现在的我和四年前的我差别不会太大，只是现在会去实践，去学习，虽然还存在很大的成长空间，但是学习、思考，这种行动带来的价值是确定的，剩下的都交给时间。

llog-81-20170913-8 年后的我该在哪
昨晚查资料时看到张鑫旭在 2012 年写的一篇文章 说说CSS学习中的瓶颈,那年我正在高二，文中他谈了自己关于 CSS 学习过程中瓶颈的观点，半夜，看的我一身冷汗。
以目前的状态而言，我处于很浮躁的一类，之前一直盲目追求热点，学习新技术，忽略基础，始终都在搬运代码。
最近用原生 Javascript 写了一个 TodoList，过程中才意识到很多基础内容完全没有印象，羞于声张，只好默默去翻文档，才反应过来竟然差了这么多。
张鑫旭毕业 8 年，8 年后的我该在哪。
